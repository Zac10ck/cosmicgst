GST Portal Submission Instructions
Export Data

Click the "Export Excel" button above to download the GSTR-1 Excel file with all invoice data.

Login to GST Portal

Go to www.gst.gov.in and login with your GSTIN credentials.

Navigate to Returns

Services → Returns → Returns Dashboard → Select Period → GSTR-1 → Prepare Online/Offline

Upload Data by Section
  • B2B Invoices: Go to "4A - B2B Invoices" - Upload invoices to registered dealers (with GSTIN)
  • B2C Invoices: Go to "7 - B2C (Others)" for intra-state sales, or "5 - B2CL" for inter-state sales above Rs. 2.5 Lakh
  • Credit Notes: Go to "9B - Credit/Debit Notes (Unregistered)"
  • HSN Summary: Go to "12 - HSN-wise Summary"
Submit Return

Review all sections → Preview → File GSTR-1 with DSC/EVC

Monthly Filers: 11th of the following month
Quarterly (QRMP): 13th of the month following the quarter
B2B vs B2C Classification:
B2B: Sales to registered dealers (customer has GSTIN)
B2C: Sales to consumers/unregistered dealers (no GSTIN)
